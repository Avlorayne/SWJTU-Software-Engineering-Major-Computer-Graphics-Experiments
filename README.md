## 简介
**西南交大软件工程系 计算机图形学实验代码**
## 使用方式
通过**CMake**构建项目会比较方便，开箱即用，唯一需要的环境是**MinGW**。不建议通过控制台挨个编译，会很麻烦。 
推荐使用 **JetBrains CLion**，下载使用会帮你把MinGW工具自动配好。
## 实现框架
**GDI**，即Windows原生窗口框架。
## 注意
若使用**Visual Studio**开发，，很可能会编译报错。原因是，Visual Studio 默认使用**Windows SDK**环境，而此处使用的是**MinGW**；二者在代码框架上有不可忽视的区别，但核心实现的原理相同，二者可轻易互通。
